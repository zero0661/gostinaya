# Гостиная — После логина

«Гостиная» — это веб-приложение для проекта «После логина».

Планируемые возможности:

- публикация откликов читателей;
- тематические обсуждения;
- поиск по сообщениям;
- модерация;
- авторизация пользователей;
- интеграция с основным сайтом milenin.pro.

## Статус

🚧 Проект находится в активной разработке.
